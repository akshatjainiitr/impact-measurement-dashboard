# User Guide

## Executive Overview

Start with the Impact Efficiency Score. It tells the Executive Director how many children reach competency per rupee invested. Use Cost per Impact to decide whether program efficiency is improving.

## KPI Tree Explorer

Click nodes across Activities, Outputs, Outcomes, and Impact to understand how field work creates measurable impact. The selected node updates the narrative panel.

## Learning Outcomes

Use the competency trend to inspect momentum. Use pre/post distributions to see whether the program is shifting learners out of lower score bands.

## Demographic Reach

Review gender, age, and state distribution to detect under-service. A reach metric is not enough; compare reach with competency rate.

## Geographic Impact

Hover district points on the India map to inspect district impact score. Use the ranked list to identify the best proof points and weakest districts.

## Data Quality Center

Use the quality score and issue list before presenting results to funders or trustees. This makes the dashboard suitable for M&E review rather than vanity reporting.

## AI Insights

Insights are generated from the same analytics tables. If no OpenAI-compatible API key is configured, the app uses deterministic insight generation so the feature remains functional.

## Scenario Simulator

Move the budget slider to estimate additional beneficiaries, learning outcomes, and competency impact. The recommended program is based on observed cost per impact.

## Forecasting

Forecasts use trend-based time-series modeling to project impact score and competency achievement. Prophet can be added in the ETL layer where the deployment environment supports it.
